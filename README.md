# 360 Revived

**360 Revived** is a project dedicated to recreating the **Xbox 360 TU19** version of **Minecraft: Legacy Console Edition** as accurately as possible.

The current goal is a **1:1 recreation** of the original Xbox 360 experience, preserving the look, feel, and quirks of the classic console edition. While the project's direction may evolve over time, authenticity remains the primary focus.

Please keep in mind that this project is developed over time, and some features are still being refined.

## Reporting Bugs

## Features

* ✅ TU19 Tutorial World
* ✅ Classic World Size set to Default
* ✅ Festive Mash-up Pack
* ✅ Summer of Arcade Skin Pack
* ✅ Xbox 360 Edition UI
* ✅ Original end credits and controller icons
* ✅ Xbox 360-style render distance
* ✅ Removed version number from the top-left corner
* ✅ DLC loading and saving fixes
* ✅ Many additional small fixes and improvements

More features and fixes will be added as development continues.

## Controls (Keyboard & Mouse)

- **Movement**: `W` `A` `S` `D`
- **Jump / Fly (Up)**: `Space`
- **Sneak / Fly (Down)**: `Shift` (Hold)
- **Sprint**: `Ctrl` (Hold) or Double-tap `W`
- **Inventory**: `E`
- **Chat**: `T`
- **Drop Item**: `Q`
- **Crafting**: `C` Use `Q` and `E` to move through tabs (cycles Left/Right)
- **Toggle View (FPS/TPS)**: `F5`
- **Fullscreen**: `F11`
- **Pause Menu**: `Esc`
- **Attack / Destroy**: `Left Click`
- **Use / Place**: `Right Click`
- **Select Item**: `Mouse Wheel` or keys `1` to `9`
- **Accept or Decline Tutorial hints**: `Enter` to accept and `B` to decline
- **Game Info (Player list and Host Options)**: `TAB`
- **Toggle HUD**: `F1`
- **Toggle Debug Info**: `F3`
- **Open Debug Overlay**: `F4`
- **Toggle Debug Console**: `F6`
